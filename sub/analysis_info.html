    시험결과 송부

  • 인터넷 신청 : 장비활용포털

    https://use.kbsi.re.kr
    • 포털검색창(분야별 장비검색, 지역별 센터장비) 또는 Quick Search 또는 장비검색 툴에서 장비명 검색
    • 장비명/ 분석전문분야/ 부서명/ 담당자/ 이용수가/ 인터넷 신청여부 확인

  • 장비활용포털 > 분야별안내전화 > 분석지원상담센터 분야별 상담위원과 상담

    분야별 안내전화

  • 14-3333(수신자부담) 를 통해 분야별 전문가와 상담

  • 국내 연구장비 검색 : ZEUS 장비활용종합포털

    www.zeus.go.kr

  • 회원가입 (소속기관 코드 입력 필수)

    회원가입 바로가기
    • 비회원 : 보유기기, 이용안내, 알림마당
    • 회원 : 보유기기, 이용안내, 알림마당, 기기예약, 마이페이지
    • 회원가입 시 기기예약 및 마이페이지 활용가능

  • 로그인 : ID / PW

    로그인 바로가기

  • 인터넷 신청 : 장비활용포털

    https://use.kbsi.re.kr
    • 기기검색
    • 장비선택
    • 스케줄 클릭
    • 하단 이용신청 클릭
    • 분석지원 예약신청 페이지
    • 신청서 작성
    • 인터넷 예약 접수중인 내용은 [마이페이지] 메뉴에서 [인터넷접수대기] 클릭

  • 직접방문, E-MAIL, 우편, 전화

    • 이용신청은 기기담당자와 반드시 사전협의 후 신청 / 이용신청 후, 담당자 접수처리 전에는 예약현황에 미반영

  • 시료 전달 주소 검색

    별도지점안내(센터주소) 시료전달 주소목록

    01

    • 장비활용 포털
    • 알림마당
    • 별도지점 안내

    02

    • 장비활용 포털
    • 분야별안내전화
    • 시료전달주소목록

  • 분석진행현황 확인 : 마이페이지 > 기기이용조회

    마이페이지 기기이용조회
    • 분석진행
    • 분석완료
    • 이용완료
    • 결제대상
    • 결제완료상 XO 확인

  • 해당 기기 담당자 분석완료 통보(전화 또는 E-mail)

  • 담당자 확인

    분야별 안내전화
    • 장비활용 포털
    • 분야별 안내전화
    • 지역별담당자에게 문의

  • 전화카드결제, 직접 방문카드결제 (KVP 안심카드결제)

  • 무통장입금 : 가상계좌 입금

  • 온라인 카드결제

    • 로그인
    • 마이페이지
    • 결제이용건 선택 후
    • 결제 아이콘 클릭

  • 검사결과 확인

    마이페이지 바로가기
    • 마이페이지
    • 기기이용 조회
    • 진행상황
    • 상세정보

  • 직접방문, E-MAIN, FAX, 우편, 웹다운로드

  • 이용료 납입 확인 후 소정양식 (시험결과서, 시험검사기록서)로 검사 결과 송부
